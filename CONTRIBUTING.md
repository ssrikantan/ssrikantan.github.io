# Contributing to This Blog

Thank you for your interest in contributing! Here's how you can help:

## Ways to Contribute

### 1. Reporting Issues
- Found a typo or error? Open an issue!
- Spotted outdated information? Let me know!
- Broken links? Please report them!

### 2. Suggesting Topics
Have an idea for a blog post topic? Open an issue with:
- Topic title
- Brief description
- Why it would be valuable to readers

### 3. Pull Requests
For corrections or improvements:
1. Fork the repository
2. Create a new branch (`git checkout -b fix/typo-correction`)
3. Make your changes
4. Commit (`git commit -am 'Fix typo in Azure AD post'`)
5. Push (`git push origin fix/typo-correction`)
6. Open a Pull Request

## Code of Conduct
Be respectful, constructive, and inclusive.

## Questions?
Feel free to open an issue or contact me directly.
